Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Marca Texto

Podemos criar também um marca texto usando a tag <mark>

Texto Grande e Pequeno

Estamos criando um texto Grande e um texto Pequeno. gat texto grande <big> tag texto pequeno <small>

Texto Deletado

Podemos marcar um texto como excluídos para indicar que ele deve ser lido, mas não considerado, <del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, tag <ins>

Existe também o sublinhado com a tag <u> (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo x2 + 3 tag <sup> para "elevar" alguma coisa

Texto Subscrito

Para inserir coisas do tipo H2O tag <sub>
